# 🇮🇹 Discover Italy

A simple and responsive travel landing page created as part of my web development learning journey. This project showcases a fictional travel website promoting beautiful destinations across Italy, including Rome, Central Italy, national parks, and the Italian islands.

## 📖 About The Project

Discover Italy is a beginner-friendly HTML and CSS project designed to practice building modern web pages. The website features attractive travel imagery, destination highlights, travel packages, and itinerary recommendations for visitors interested in exploring Italy.

## ✨ Features

- Responsive travel landing page
- Hero section with destination introduction
- Travel package showcase
- Top itinerary recommendations
- Clean and modern layout
- Beginner-friendly code structure

## 🗺️ Featured Destinations

- Rome & Central Italy
- Nature & National Parks
- South Italy & Islands

## 🛠️ Built With

- HTML5
- CSS3

## 🎯 Learning Goals

This project helped me practice:

- Structuring web pages with HTML
- Styling layouts with CSS
- Working with images and content sections
- Creating visually appealing landing pages
- Building projects for my developer portfolio
